# カラーテーマ提案

現在の青いグラデーション背景を、より見やすいデザインに変更する提案です。

## 現在のデザイン

```css
background: linear-gradient(135deg, #001d6c 0%, #0043ce 50%, #0f62fe 100%);
color: #fff;
```

**問題点**:
- 濃い青色で目が疲れやすい
- 白文字とのコントラストが強すぎる
- 長時間の閲覧に向かない

---

## 提案1: ライトモダン（推奨）

明るく清潔感のあるデザイン。最も読みやすく、長時間の閲覧に適しています。

### カラーパレット
- **背景**: 明るいグレー `#f5f7fa`
- **カード背景**: 白 `#ffffff`
- **テキスト**: ダークグレー `#2c3e50`
- **アクセント**: IBM Blue `#0f62fe`
- **ボーダー**: ライトグレー `#e1e8ed`

### 特徴
- ✅ 目に優しい
- ✅ 長時間閲覧に最適
- ✅ プロフェッショナルな印象
- ✅ 高いコントラスト比（WCAG AA準拠）

### プレビュー
```
┌─────────────────────────────────────┐
│ 背景: 明るいグレー (#f5f7fa)         │
│ ┌─────────────────────────────────┐ │
│ │ カード: 白 (#ffffff)             │ │
│ │ テキスト: ダークグレー (#2c3e50) │ │
│ │ [ボタン: IBM Blue (#0f62fe)]    │ │

└─────────────────────────────────────┘
```

## 提案2: ソフトグラデーション

柔らかいグラデーションで、現代的な印象を保ちつつ見やすさを向上。

### カラーパレット
- **背景**: 淡いブルーグレーのグラデーション
  - `#e8f4f8` → `#d6e9f5` → `#c4dff2`
- **カード背景**: 白 `#ffffff` (半透明)
- **テキスト**: ダークブルーグレー `#1a3a52`
- **アクセント**: ティールブルー `#0891b2`
- **ボーダー**: ライトブルー `#b8d4e8`

### 特徴
- ✅ 柔らかい印象
- ✅ 目に優しい
- ✅ モダンなデザイン
- ✅ IBM Bobのイメージを保持

## 提案3: ダークモダン

ダークモードスタイルで、目の疲れを軽減しつつスタイリッシュな印象。

### カラーパレット
- **背景**: ダークグレー `#1a1d23`
- **カード背景**: ダークブルーグレー `#252a34`
- **テキスト**: ライトグレー `#e4e6eb`
- **アクセント**: ブライトブルー `#4a9eff`

### 特徴

- ✅ 夜間の閲覧に最適
- ✅ モダンでスタイリッシュ
- ✅ バッテリー消費が少ない（OLED画面）

## 推奨

**提案1: ライトモダン** を推奨します。

### 理由
1. **最も読みやすい**: 黒文字×白背景は最高の可読性
2. **アクセシビリティ**: WCAG AA基準を満たす高コントラスト
3. **プロフェッショナル**: ビジネス向けサイトに最適
4. **汎用性**: どんな環境でも見やすい
5. **実装が簡単**: 複雑なグラデーション不要
